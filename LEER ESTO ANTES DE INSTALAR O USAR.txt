                                                   Instrucciones


1 Instale los siguientes programas en el orden marcado.
	1.1 Java
	1.2 Framework y Visual (sólo si está usando Windows XP)
	1.3 MySQL
		1.3.1 Seleccione la carpeta "Instalador MySQL"
		1.3.2 Ejecute el archivo: "mysql-installer-commercial-5.6.26.0.exe"
		1.3.3 La contraseña de la cuenta de usuario root debe ser: t1nocota+  de lo contrario no funcionará el sistema
		1.3.4 Regrese a la carpeta MySQL, ejecute el archivo: "mysql-administrator-1.1.9-win.exe"
			e instale el programa MySQL Administrator.

2 Buscar MySQL Administrator
	2.1 Diríjase al botón de inicio
	2.2 Seleccione la opción "Todos los programas"
	2.3 Busque la carpeta MySQL y selecciónela
	2.4 Abra el programa

3 Abra el manual de instalación de la carpeta correspondiente y siga las indicaciones para configurar MySQL Administrator,
      cargar los datos e instalar el Sistema de Inventario de Bienes
